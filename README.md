# What is this library?

This library is an extension of [`Kruskal-AlmostFull`](https://github.com/DmxLarchey/Kruskal-AlmostFull).
It contains a detailed constructive/inductive proof of Wim Veldman's variant of the proofs of
Higman's and Kruskal tree theorems.
